# Analiza in modeliranje temperature z uporabo WeatherData

## Opis projekta

V tej projektni nalogi bom razvila aplikacijo v programu **Wolfram Mathematica**, ki uporablja funkcijo `WeatherData` za pridobivanje vremenskih podatkov. Cilj projekta je analizirati in vizualizirati vremenske podatke. 


Aplikacija bo uporabniku omogočala izbiro mesta in časovnega intervala ter prikaz gibanja temperature skozi čas. Poleg tega bo implementiran matematični model za aproksimacijo temperature, ki temelji na periodičnih funkcijah (sinusni model), saj se temperatura skozi leto spreminja ciklično.

## Cilji projekta

 
- obdelava podatkov z uporabo `WeatherData`
- prikaz temperature skozi čas
- modeliranje temperature s sinusno funkcijo
- primerjava podatkov z matematičnim modelom
